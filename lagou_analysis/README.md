# 2019年8月数据分析岗位需求现状分析
****
## 说明
1. 数据来自2019-08-14拉勾网搜索关键词“数据分析”的职位列表，爬取了依职位数量降序排列的前10个城市,1886条记录。
2. 由于拉勾一次只显示30页×15个职位，故对超过450个职位的城市，降序爬取数量超过50个职位的一级行政区。
3. 本文使用pandas进行数据处理和分析，结合seaborn和pyecharts包进行数据可视化。Python运行版本为3.6.7。
4. 主要的文件为：  
  * lagou_analysis.ipynb：Jupyter notebook代码，对拉勾数据分析职位招聘状况进行分析
5. 数据：  
  * lagou0814_total.csv

## 分析目标
* 由于近期致力于找一份数据分析相关的工作，故想通过数据考察一下市场对数据分析师的需求状况，以及筛选出适合尝试的公司和职位。

## 数据清洗提要
爬取了13个字段，共1879条不重复记录。  
![数据基本情况](https://github.com/zengkewan/python_DA/raw/master/lagou_analysis/数据基本情况.png)

由于原数据‘工资’字段给出的都是薪资范围，而且很多20k-40k这种大范围的写法，所以取了给出工资范围的下界作为工资的参考值，增加了‘工资下界’这个字段存储以元为单位的工资参考值。  
![工资下界](https://github.com/zengkewan/python_DA/raw/master/lagou_analysis/工资下界.png)

## 数据分析
#### 提出问题
1. 城市与职位
2. 学历与职位
3. 工作经验与职位
4. 公司与职位
5. 职位发布时间

#### 城市与职位数量

北上广深杭这五个城市毫无疑问地占据了前五的位置，其中北京有740个职位，占39.38%；上海有398个，占21.18%；北上广深杭这五个城市占了92.33%，说明这个职位还是集中在一线城市。

如果把数据爬全，前5个城市的职位数量占总体的比例应该会比目前这个数低几个百分点，但是影响也不会很大。

另外值得注意的是，深圳排到了第三，在广州和杭州的前面，而且高出的比例不低。  
![城市与职位数量](https://github.com/zengkewan/python_DA/raw/master/lagou_analysis/城市与职位数量.png)

#### 城市与薪资

以城市分组，看不同城市平均工资怎么样，发现薪资前五名中，厦门竟然在列，不过厦门样本量太小，数据不具有代表性，还是北上广深杭的天下。  
![各城市平均工资](https://github.com/zengkewan/python_DA/raw/master/lagou_analysis/各城市平均工资.png)

通过小提琴图看薪资分布，从图中看，开出行业顶级高薪的，是北上深杭。  
![城市与薪资分布](https://github.com/zengkewan/python_DA/raw/master/lagou_analysis/城市与薪资分布.png)

#### 学历与职位数量

要求至少本科毕业的数量是最多的，约占83.6%。  
要求博士以上的最少，毕竟相对于机器学习和数据挖掘之类的岗位，数据分析算是对技术要求相对较低的。  
写学历不限的，多半是字段里填不限，但是在详情里要求了学历。  
![数据分析岗位学历要求](https://github.com/zengkewan/python_DA/raw/master/lagou_analysis/数据分析岗位学历要求.png)

数据分析行业在中国快速发展，也就是近四五年以来的事情，资深从业者的数量累积有限，博士学历的资深从业者更是寥寥。  
![学历与薪资箱形图](https://github.com/zengkewan/python_DA/raw/master/lagou_analysis/学历与薪资箱形图.png)

#### 工作经验与职位数量

要求有一年以上经验的加起来占九成，要求3-5年的最多。  
应届毕业生和一年以下的只占8.1%，表明了公司想要又相对年轻又有经验的熟练工。  
而那10.8%不限的，还是那句老话，基本上都在详情里要求了经验，不是1-3年，就是3-5年。  
![数据分析岗工作经验要求](https://github.com/zengkewan/python_DA/raw/master/lagou_analysis/数据分析岗工作经验要求.png)

看看分布，每一档上面都有离群值，意味着一定程度的能者多得，一小部分特别有才华的，可以拿到高出均值好几倍的薪资。  
从这个箱形图也可以看到，经验不限是假的，总体来说很接近1-3年的经验要求。

![公司规模与职位数量和薪资](https://github.com/zengkewan/python_DA/raw/master/lagou_analysis/公司规模与职位数量和薪资.png)

#### 职位发布时间

这就很值得玩味了，我8月14号爬的数据，‘发布时间’为当天的竟然数量最多，比例超群。
![发布时间与职位数量](https://github.com/zengkewan/python_DA/raw/master/lagou_analysis/发布时间与职位数量.png)

我猜测这个‘发布时间’并非职位首次发布的时间。  
于是我8月18号又爬了‘数据分析’关键词在北京的职位列表。  
可以看到相同的公司、职位，过了几天，发布时间集体变了，而且精确到秒全都一模一样。  
所以企业端很可能有一个【一键重新发布】之类的按钮。  
![发布时间对比](https://github.com/zengkewan/python_DA/raw/master/lagou_analysis/发布时间对比.png)

我又调查到，拉勾把职位收藏以后，可以看到职位第一次发布的时间，我搜了刚刚那个公司的职位，把搜索页截图那几个职位收藏了：   
![搜索页发布时间](https://github.com/zengkewan/python_DA/raw/master/lagou_analysis/搜索页发布时间.png)  
![收藏页发布时间](https://github.com/zengkewan/python_DA/raw/master/lagou_analysis/收藏页发布时间.png)

这就意味着，有部分是改一改之前发布的职位重新发布一下省事，有部分职位可能是公司用来挂曝光度的。    
尽管如此，就算‘发布时间’字段并非真正首次发布的时间，但是至少说明有人为操作的痕迹，只是降低了有效职位比例。


## 总结
从以上的数据分析我们可以知道：

1. 数据分析职位的招聘，主要集中在“北上深杭广”这5个大城市（占九成），北京以740个岗位（39.4%）遥遥领先。
2. 平均薪资为14520元人民币，“北上广深杭”这五个城市中，北京的薪资最高，再者是上海、深圳。
3. 学历要求以本科居多83.6%，薪资和学历基本上是正相关的关系，博士最高，接着是硕士、本科、大专。
4. 工作经验要求3-5年的最多（41.6%），1-3年的次之（25.0%），而应届毕业生和一年以下的加起来只占8.1%，都表明了公司想要相对年轻又有经验的熟练工。
5. 工作经验与薪资呈正相关，应届毕业生和1年以下的在6k左右，从1-3年开始就是11k往上了。
6. 在招职位数量前15的公司，多数是大家熟悉的知名公司，比如字节跳动、腾讯、美团点评、拼多多、京东集团等。
7. ‘公司规模’2000人以上的公司发布的职位总数量最多，呈正相关，这方面的需求还是以中大型企业为主，薪资总体来说也是大型企业给的高。
8. 关于职位发布时间的分析，告诉我们有效职位的比例比预想的低。
